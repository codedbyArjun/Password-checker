# 🔐 Password Strength Checker

This is a simple Python project that checks how strong a password is.

I built this while learning Python to get more comfortable with working with conditions, functions, and strings. The program checks a password based on a few basic rules and then tells whether it's weak, medium, or strong.

## Features

- Checks password length
- Checks for uppercase letters
- Checks for numbers
- Checks for special characters
- Gives simple feedback about password strength
